Exercise 5: Create Text Files with touch   

 touch command: allows you to change last access and last modification dates. without any options, you can create an empty file. If the file exists, only the dates will be "touched".

  1. Change to /etc directory and try to create a new file. What happened? Were you able to create the file? Why or why not?
  2. ~/lab5$ cd /etc
    /etc$ touch file5
  3. Although you can use touch to create empty files, what is the main function of the touch command?
  4. Scenario: you are preparing to distribute a set of files-music, software, whatever-and you want them to all have the same time and date; change to the directory that holds the files and enter:
  5. ~/lab5$ touch *
  6. All the files matched by the * wildcard now have the same modification time and access time. To change the modification time only, use the -m option. To change the access time only, use -a option. To use a specific time and date instead of the current time, use -t followed by the format [[YY]YY]MMDDhhmm[.ss]. See manual page for more details.
  7. By default, ls command displays the modificatin time. How can you view the access date and time?
  8. You can create more than one new file at a time
  9. ~/lab5$ touch data1 data2 data3 temp extra
  10. When you are updating date and time for a number of files, and you don't want touch to create any new files, use the -c (no create) option:
  11. ~/lab5$ touch -c backup1 backup2 backup3 backup4
