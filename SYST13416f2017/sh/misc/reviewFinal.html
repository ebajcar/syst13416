Course Learning Outcomes

  1. 1. Summarize the history of the UNIX Operating System
  2. 2. Compare characteristics of UNIX with other operating systems
  3. 3. Use command-line and GUI tools to interact with the OS
  4. 4. Illustrate how UNIX manages files and directories
  5. 5. Restate main differences between major popular shells
  6. 6. Use file system tools
  7. 7. Obtain information about other users and the user environment
  8. 8. Customize individual user environments
  9. 9. Use ASCII text editors to create data file and scripts
  10. 10. Use redirection, pipes and filters to make commands more efficient
  11. 11. Solve simple problems by programming in shell scripting languages
  12. 12. Monitor LINUX / UNIX processes
