Assignment 4 SYST13416 Introduction to Linux Operating System

Write a bash script called compute (no extension) that will calculate addition, subtraction, multiplication, and division of two numbers (real and integer). If both numbers are integers, return an integer result, if one of the values is a real, return real number with default precision (unless specified, see below). Note the order of arguments. Examples:

compute 7 + 3

compute 9.37 / 7.21

compute 100 x 2.21

compute 2783.34 – 25

Allow also calculation of two functions, hypotenuse (see earlier handout) and area of circle (given radius). These calculations will always be real, even if input is integer. The –f option token is followed by option argument which specifies function is to be calculated. Arguments (values) are listed last.

compute –f hyp 8.1 3.5

compute –f area 4.21

compute –s 5 9.37 / 7.21

compute –s 9 –f hyp 8.1 3.5

compute –f hyp –s 3 8.1 3.5

Include all error checking and handling. Include all appropriate inline documentation. Provide user help how to use the utility either when the sequence, value, or number of arguments is incorrect, or when user asks:

compute –h

compute –help

Design and implement an interactive front-end web app that will allow the user to choose what they want to compute.

  1. Make it user friendly and validate the input using the pattern= attribute. Be sure to use a well-formed and valid HTML5 and CSS3.
  2. On form submit, use a PHP program to process the user input. Use the GET method to pass the key=value pairs.
  3. Validate the input again (using PHP regex functions) on the server

  4. Construct the appropriate command string to pass to the exec function.

  5. Call your bash script from the PHP program to do the calculation. An example:

    $output = exec('compute –s 5 9.37 / 7.21');

    echo "<p>$output</p>"

    AN ASIDE: you can execute a shell command from PHP, such as: $curPath = shell_exec('pwd');

Make sure that each and every file is well documented, including the file relationships and dependancies.

Publish on dev.fast.sheridanc.on.ca, in public_html/syst13416. All files can be placed in this single directory (the html, php, and script).
