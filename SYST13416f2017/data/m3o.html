Your learning objectives in this module are:

  • Illustrate how Linux manages files and directories.
  • Describe the role of partitions, master boot record, and partition table.
  • Explore the difference among file system types (ntfs, fat32, ext3, ...)
  • Define the differences and similarities between directories and folders.
  • Sketch the parent-child relationship of directories and their contents.
  • Use shortcuts for the current working directory, the parent, the root, and user's home directory.
    • Construct a directory tree to organize a Linux account.
  • Distinguish between relative and absolute pathnames.
  • Navigate a directory tree.
  • Duplicate and rename directories.
  • Delete directories without removing entire account.
