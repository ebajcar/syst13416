Midterm Review

— SYST13416 Introduction to Linux Operating System —

Exam Information

  • The midterm exam covers the covered material to date, specifically,
    • Module 1: The Essence of Unix
    • Module 2: Linux Command Line
    • Module 3: Linux File System
    • Module 4: Linux Files
    • Module 5: Processes and Utilities
  • All questions are answered on test paper. Write clearly and submit ALL papers provided.
  • The midterm exam is a closed book test and therefore you have no access to your computer or any on-line material.
  • You are permitted to use your handwritten, original notes! (no reproductions, no printouts, no images, no photocopies). YOU KEEP YOUR NOTES, DO NOT SUBMIT with your exam.
  • If you are caught with any reproduced material, ALL notes will be taken away and your exam mark may be a subject to penalty! (review Academic Honesty policies)
  • The length of the exam is 90 minutes. The maximum number of marks is 50. Breakdown:
    1. about 1/3 multiple choice and true-false questions,
    2. about 1/3 short answers and
    3. about 1/3 writing code.

    cat bananas apples > fruit4

Sample Questions:

  1. True or False? A process is the operating system execution of a program. When you log in, the process running is normally a shell, called a login shell. When you enter a command at the prompt, the shell has the responsibility of finding the command and arranging for the command to be executed. This is done with system calls. A system call is a request for kernel services.
  2. True or False? A utility performs a task that is frequently related to the operating system. A utility is simpler than an application program, although no clear line separates the two. Linux distributions include many utilities. You can also download many utilities from the Internet. Examples of utilities are cp (copies a file), ls (lists information about files), and ssh (securely connects to a remote computer).
  3. This command displays all processes you have currently running, or stopped, which are associated with your terminal:
    1. fg
    2. bg
    3. jobs
    4. top
    5. free
