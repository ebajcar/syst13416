Features

jCover is a jQuery plugin to create a flat designed paper folding effect at the bottom-right corner, which is useful for both stylishness and a hint for a link.

It uses CSS border tricks to achieve the effect, without using any image, which enables us to create corners of any size.

It's a tiny library with minified JavaScript file of 818 bytes and CSS file of 377 bytes.
