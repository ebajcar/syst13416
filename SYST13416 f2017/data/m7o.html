Your learning objectives in this module are:

  • Understand how to write a basic regular expression.
  • Search for text in a file or a group of files in the same directory with a regular expression.
  • Use modifiers: asterisk (*), period (.), plus(+), and question mark (?)
  • Use anchors: hat (^), dollar sign ($), word boundaries ( \< and \> )
  • Use character sets.
  • Use escape character (\).
  • Use Linux commands with regular expression.
