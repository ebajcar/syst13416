Introduction

This course provides an introduction to the Linux operating system. It is not aimed to be a survey course. Students acquire an indepth understanding and appreciation of the Linux programming environment by exploring the capabilities of a single shell environment; students create shell scripts, use scripting tools, and interact directly with the operating system environment, to demonstrate their mastery of key concepts.

The objective of the course is to give students a solid foundation of the operating system environment, to develop lifelong skills, including critical thinking, problem solving, communication and interpersonal skills, and covers the following key concepts:

  • Developer's shell environment--the command prompt.
  • File systems, text file manipulation, text editors
  • Interactive and batch shell programming.
  • Linux utilities and security.

Why Linux?

 

Learning Outcomes

By the end of this course, you will have demonstrated the ability to:

  1. Summarize the history of the UNIX Operating System
  2. Compare characteristics of UNIX with other operating systems
  3. Use command-line and GUI tools to interact with the OS
  4. Illustrate how UNIX manages files and directories
  5. Restate main differences among major popular shells
  6. Use file system tools
  7. Obtain information about other users and the user environment
  8. Customize individual user environments
  9. Use ASCII text editors to create data file and scripts
  10. Use redirection, pipes and filters to make commands more efficient
  11. Solve simple problems by programming in shell scripting languages
  12. Monitor LINUX / UNIX processes
