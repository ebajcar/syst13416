Preparing for Midterm

What's on the Midterm?

Learning objectives
  • Summarize the history of the UNIX Operating System
  • Compare characteristics of UNIX with other operating systems
  • Use command-line to interact with the OS
  • Illustrate how UNIX manages files and directories
  • Restate main differences between major popular shells
  • Use file system tools
  • Use ASCII text editors to create data file and scripts
  • Use redirection, pipes and filters to make commands more efficient

What's on the Midterm?

Content
  • Covers Modules 1, 2, 3, and 4.
  • Covers Worksheets 1, 2, 3, 4, and 5
  • Covers concepts and skills related to the learning objectives (see previous slide).

What's the format?

  • Online SLATE Quiz with Respondus lockdown.
  • Exam opens at the beginning of regular class (week 7)
  • It will be available for 30 minutes to start.
  • You have 60 minutes to finish from the time you start.
  • One question is presented at a time. You cannot go back.
  • ~50 question — mix of true/false, multiple choice, multi-select, match, and fill-in-blanks.
  • Majority of questions come from the asssigned material and similar to quiz questions.
