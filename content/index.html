Course Information

Course Overview
SYST113416 Linux/UNIX Operating Systems

It is an introductory course to understanding how operating systems work and support computing environments to develop and run applications. In this course, the students learn about the fundamental elements of the operating system, specifically basic UNIX utilities, file systems, shell environments, shell and other scripting languages, operating system security, and standard text editors. The knowledge and skills acquired in this course will be used. This course will meet program objectives in the area of Operating Systems and Applications, specifically: 1. Operating systems concepts including organization, architectures, role and environments. 2. Exposure to operating systems on different platforms.

Course Learning Outcomes

1Summarize the history of the UNIX Operating System
2Compare characteristics of UNIX with other operating systems
3Use command-line and GUI tools to interact with the OS
4Illustrate how UNIX manages files and directories
5Restate main differences between major popular shells
6Use file system tools
7Obtain information about other users and the user environment
8Customize individual user environments
9Use ASCII text editors to create data file and scripts
10Use redirection, pipes and filters to make commands more efficient
11Solve simple problems by programming in shell scripting languages
12Monitor LINUX / UNIX processes
